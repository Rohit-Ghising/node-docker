# MERN Dockerized Portfolio with Jenkins CI/CD

A full-stack MERN portfolio application containerized using Docker and automated with Jenkins CI/CD pipeline.

---

# Tech Stack

## Frontend
- React
- Vite
- Typescript

## Backend
- Node.js
- Express.js
- MongoDB

## DevOps & Deployment
- Docker
- Docker Compose
- Nginx
- Jenkins
- Dockerhub
- Images
